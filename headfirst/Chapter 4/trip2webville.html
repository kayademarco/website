  1. What solution are we using to the problem presented in the chapter of finding yourself a hosting company? What are the advantages and disadvantages of our solution? What alternatives could we have used, and which alternative did we use earlier in the year?
    • In this class we have solved the problem of finding a host by using the platform GitHub. This platform allows us to publish websites with our own domain with their "watermark" of sorts.
    • The advantages of using GitHub are that you can easily share your website and code with others. This helps us when collaborating with others. Another advantage is that it allows for version control which allows you to branch off and try new things without having a lot of collateral damage.
    • The disadvantage of GitHub is that you can not host a commerce page or anything that requires back-end communication (forms, chat, etc)

    • They used GoogleDomains which is a service by google

    • The workflow we use on GitHub allows us to "revert" back to older versions of out website, whereas, the FTP workflow presented in the textbook does not allow you to "revert" back to older versions.
